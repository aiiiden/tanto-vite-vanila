## STEP-BY-STEP

1. **Install Dependencies**:

   ```bash
   npm install
   ```

2. **Run the Application**:

   ```bash
   npm run dev
   ```

3. **Build the Application**:
   ```bash
    npm run build
   ```

Then open the `dist` folder. -> This is the folder that contains the built application.

4. **Run the Application**:

   ```bash
   npm run preview
   ```
